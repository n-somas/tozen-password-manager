# TOZEN Password Manager

Offline password manager developed in Java.

## Features
- User registration and login
- Encrypted local password storage
- AES-256-GCM encryption
- PBKDF2-HMAC-SHA-256 key derivation
- Recovery function
- Password generator
- Offline usage without cloud

## Technologies
- Java 17
- JavaFX
- SQLite
- NitriteDB
- JUnit 5

## Note
This repository does not contain real user data, production secrets, or local database files.
